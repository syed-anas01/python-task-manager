# python-task-manager
A python based Task Manager for tracking daily to-do list via command line interface(CLI)


## 🚀 Features
* **Add Tasks:** Quickly add new items to your list.
* **View Tasks:** See all your current pending tasks in a clean format.
* **Auto-Save:** Your tasks are saved locally so you don't lose them.
* **Privacy:** Uses `.gitignore` to keep your personal data files off GitHub.

## 🛠️ How to Run
Make sure you have Python installed, then run:
```bash
python todo_list.py
